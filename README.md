# FloatDraw: The Swiss Army Knife of Screen Annotation Tools

A powerful screen annotation tool built with Electron, featuring drawing tools, shapes, stickers, and more.

## Features

-   Basic Shapes
    -   Rectangle
    -   Circle
    -   Triangle
    -   Lines (Straight and Freehand)
    -   Any Polygon
-   Stickers: Pre Added Shapes
    -   Star
    -   Cross
    -   Tick
    -   Single and Double Sided Arrows
-   Highlighter
-   Laser Pointer
-   Screenshot Tool
-   Text Support
-   Backgound Pages
-   Custom Color Picker &nbsp;
-   And More ...

## Installation

**Packages can be Found in the [releases](https://github.com/shnwz3/floatdraw/releases) Section**

**Note**: Imagemagick is required for screenshots on linux and screenshots are not guaranteed to work on wayland as of now.

| OS                 | Instructions                                                                                                                          |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------- |
| Debian Based Linux | A .deb archive has been provided                                                                                                      |
| Arch Linux         | Install the prebuilt package archive provided in the releases section                                                                 |
| Other Linux        | An AppImage has been provided                                                                                                         |
| Windows            | Download and run the setup provided in the releases section. You may need to disable your antivirus and / or mark the setup as safe   |
| Mac OS             | A dmg has been provided                                                                                                               |
